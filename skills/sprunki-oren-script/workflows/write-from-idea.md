# Workflow: Viết Script từ Ý tưởng / Outline (Adult Version)

## Đầu vào

- **Ý tưởng 1 câu**: "Oren cần trả rent cuối tháng"
- **Outline sơ bộ**: Danh sách các cảnh
- **Tên episode**: "Oren's Rent Is Due"

## Prerequisites

- Đọc `SKILL.md` (world rules, character roster — ADULT VERSION, storytelling DNA, voice profiles)
- Đọc `templates/shot-format.md` cho output format
- Đọc `references/voice-profiles.md` cho dialogue voice cues

## Quy trình

### Bước 1: Xác định Episode Structure

**Single Episode:**
```
WANT → BROKE! → HUSTLE → [FAIL/COMPLICATION] → GET PAID → ENJOY!
```

**Compilation (video dài):**
```
[Mini-arc 1: WANT→HUSTLE→ENJOY] → [Kết nối nhân quả] → [Mini-arc 2] → ...
```

**Xác định cho mỗi arc:**
- **WANT**: Oren muốn/cần gì? (rent, date night, new gear — phải CỤ THỂ và ADULT)
- **BROKE! trigger**: Check banking app → $3.47 (signature moment bắt buộc)
- **HUSTLE**: Làm gig gì? Ở đâu? (bartending, delivery, freelance beats — ADULT jobs)
- **GET PAID**: Professional — Venmo, direct deposit, cash in envelope. "Good work, man!" + handshake
- **ENJOY**: Thỏa mãn mục tiêu — rent paid, date night, new equipment
- Locations: 2-4 **ADULT** American environments (apartment, bar, coffee shop, coworking)
- Sprunki friends: ai xuất hiện? Voice profile nào?
- Human archetypes: vai trò + reaction pattern (PEER, not mentor-to-child)
- Duration: 1-2 phút (10-15 shots) hoặc 3-10 phút compilation

### Bước 2: Xây dựng Shot List

- WS establishing → MS action → CU reaction → MS next (lặp)
- 40% MS, 20% CU, 15% WS, 15% MWS, 10% Insert
- 3-Beat Arc cho mỗi hành động: Setup → Grind → Payoff

**Human Interaction Pattern (PEER VERSION):**
```
MS Oren action → MWS two-shot (bình đẳng) → CU human reaction → CU Oren reaction
```

### Bước 3: Viết từng SHOT

Format: `// SHOT XX | duration | shot_type` + fields

**Checklist mỗi [NOTE]:**
- [ ] Camera đứng đầu câu
- [ ] Subject + vị trí rõ ràng
- [ ] 1 hành động chính duy nhất
- [ ] **ADULT** American context/environment
- [ ] Oren body language: lean, swagger, tay trong túi, headphones, coffee
- [ ] KHÔNG style descriptions
- [ ] KHÔNG ngoại hình chi tiết
- [ ] Ít nhất 1 adult context cue (coffee, beer, laptop, apartment, bar)

### Bước 4: Dialogue Layer — Voice Profile Alignment

- **[Character]** dialogue PHẢI match voice profile:
  - Oren: short, chill, slang → "Aight bet", "That's fire", "Rent's due..."
  - Pinki: warm but direct → "Did you pay rent?", "I'm proud of you"
  - Simon: hyped, fast → "BRO. Trust me.", "We're gonna blow up"
  - Durple: smooth, slow → "Hmm. Let me think about that."
  - Gray: minimal, impactful → "You already know the answer."
  - Sky: curious, fresh → "Wait, can I ask something?"

- **Humans**: casual peer English, KHÔNG patronizing
  - ✅ "Nice work, man." / "What can I get you?" / "You good, bro?"
  - ❌ "Good job, little buddy!" / "Aww, you did it!" → CHILD TALK

### Bước 5: Audio Layer

- **[SFX]** mọi shot — American **ADULT** ambient (coffee machine, bar chatter, keyboard, city traffic)
- **[Character]** dialogue: voice-appropriate per profile
- **[BGM]** lo-fi / chill hip-hop / indie, chỉ khi mood thay đổi
- **60%+ SFX-only**

### Bước 6: Review

- [ ] WANT → BROKE! → HUSTLE → GET PAID → ENJOY! đầy đủ
- [ ] $3.47 banking app signature moment có
- [ ] Oren body language đúng (**ADULT** chill, NOT kid excited)
- [ ] Human reactions dùng **PEER** patterns (bình đẳng, KHÔNG patronizing)
- [ ] Dialogue matches voice profiles
- [ ] 60%+ shots SFX-only
- [ ] ZERO text on screen
- [ ] ALL environments are adult (apartment, bar, office — NOT school, playground)
- [ ] ALL props are adult (coffee, beer, laptop — NOT toy, candy, backpack)
