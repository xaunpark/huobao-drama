## Group Dynamics (Adult Version)

```
Simon (hype man, ideas)  ←→  Oren (chill, music)  ←→  Durple (wise, calm)
                                  ↕
                             Pinki (GF, grounding)
                                  ↕
                     Gray (quiet anchor) + Sky (fresh perspective)
```

**Khi cả nhóm xuất hiện** (thường tại bar, apartment hangout, rooftop):
- Simon đề xuất kế hoạch wild
- Oren "...aight, bet" (goes along)
- Durple bình tĩnh đánh giá: "Hmm. Let me think about that."
- Gray quan sát, drop one-liner sắc bén
- Sky hỏi câu hỏi thực tế
- Pinki reality-check: "Okay but who's paying for this?"

## Adult vs Child — Key Differences

| Aspect | ❌ Child/Teen Version | ✅ Adult Version |
|---|---|---|
| **Housing** | Parents' house | Studio apartment, roommate possible |
| **Money source** | Allowance, odd jobs | Freelance gigs, side hustles, direct deposit |
| **$0 moment** | Empty piggy bank | Banking app: $3.47, past-due rent |
| **Relationships** | Crush, puppy love | Girlfriend, dating, anniversary |
| **Social** | Playground, school | Bar, coffee shop, rooftop hangout |
| **Food** | Juice box, candy | Coffee, beer, ramen, takeout |
| **Problems** | Lost toy, homework | Rent, bills, work deadlines, burnout |
| **Vocabulary** | "Bruh", "Sick!" | "Aight bet", "That's fire", "Rent's due" |
| **Body language** | Bouncy, excited | Lean, swagger, confident, contained |
| **Motivation** | Want cool thing | Need to survive + want cool thing |
