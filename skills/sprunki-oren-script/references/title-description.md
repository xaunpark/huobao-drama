# Sprunki Oren — Title, Description & Thumbnail Generator (Adult Version)

Tạo tiêu đề YouTube, mô tả video, và prompt thumbnail cho kênh Sprunki Oren.
Ngôn ngữ: **Tiếng Anh** (target audience: young adults + general audience).

---

## PHẦN 1: TITLE

### Formula

```
[EMOJI] [HOOK] — Oren [Setup]... [Cliffhanger]! [EMOJI CLUSTER]
```

### Emotion Hooks (mở đầu)

| Hook | Khi nào | Ví dụ |
|---|---|---|
| 😱 OMG | Sốc, bất ngờ lớn | "😱 OMG — Oren Had $3 Left for RENT..." |
| 🔥 INSANE | Kết quả ngoạn mục | "🔥 INSANE — Oren's Bartending Made HOW Much?!" |
| 😭 SO SAD | Cảm động | "😭 SO SAD — Oren Worked ALL WEEK Just to Surprise Pinki..." |
| 💀 NO WAY | Bất ngờ hài hước | "💀 NO WAY — Oren Actually Got a REAL Job?!" |
| 🤯 UNREAL | Twist ngoài sức tưởng tượng | "🤯 UNREAL — Broke Oren's Beat Went VIRAL!" |
| ❤️ WHOLESOME | Ấm áp, touching | "❤️ WHOLESOME — What Oren Did for Their Anniversary Will Make You CRY" |
| 😂 HILARIOUS | Hài hước chính | "😂 HILARIOUS — Oren's First Bartending Shift Was CHAOS" |

### Structure

| Thành phần | Rule |
|---|---|
| **Emoji Hook** | 1 emoji + ALL CAPS word (2-3 chars) |
| **Setup** | Oren + tình huống ADULT, dùng CAPS cho emphasis |
| **`...`** | Suspense break |
| **Cliffhanger** | Gợi ý, KHÔNG spoil. Kết bằng `?!` hoặc `!` |
| **Emoji Cluster** | 2-4 emoji cuối |

### Examples

```
😱 OMG — Oren Had $3.47 But RENT Was Due Tomorrow... His HUSTLE Was INSANE! 🍕💰🏠
🔥 INSANE — Oren Started Bartending and THIS Happened! 🍸💸
😭 SO SAD — Oren Worked ALL WEEK Just to Buy Pinki a Gift... 💝🥹
💀 NO WAY — Oren's Tax Filing Was the MOST Chaotic Thing EVER 😂📋
🤯 UNREAL — Broke Oren Made BANK Selling Beats in ONE Weekend! 🎵🤑
```

## PHẦN 2: DESCRIPTION

#### 1. Hook (2-3 lines)

```
Oren is BROKE. Again. 😭💸
Rent's due tomorrow and his bank account says $3.47.
So what does he do? He gets a BARTENDING GIG. At a PUB. 🍸
```

#### 2. Story Recap (5-8 lines)

```
Watch as Oren learns to mix cocktails from scratch 🔥

The bar manager was impressed though... and so were the customers! 🍸✨

But things don't go perfectly...
Let's just say there's a LOT of broken glass involved 🫣😱
```

#### 3. Climax Teaser (2 lines)

```
What happens when Oren finally checks his Venmo? 💰
Watch till the END to find out! 🏠✨
```

#### 4. CTA

```
💖 SUBSCRIBE for more Oren adventures!
🔔 Hit the BELL so you never miss an episode!
👍 LIKE if Oren's adult struggles are too relatable!
```

#### 5. Hashtags

Core: `#Oren #Sprunki #SprunkiOren #animation #adulting #hustle #relatable`
Thêm chủ đề: `#bartending #rent #broke #freelancer #sliceoflife`

## PHẦN 3: THUMBNAIL PROMPT

### Composition

```
[OREN ~40-50% frame] + [KEY PROP / CRISIS OBJECT] + [SIMPLE BACKGROUND]
```

### 3 Layout Patterns

| Pattern | Khi nào | Mô tả |
|---|---|---|
| **Subject + Crisis** | Bills, rent, broke | Close-up Oren, phone showing $3.47, panic expression |
| **Subject + Action** | Đang làm gig | Oren CENTER, đang bartend/cook/deliver |
| **Subject + Reward** | Get paid, enjoy | Oren with Venmo notification, satisfaction |

### Expression Rules (Adult)

| Emotion | Biểu cảm Oren |
|---|---|
| **Dread/Broke** | Eyes half-lidded then widen, jaw clench, hand on face, phone showing $3.47 |
| **Determined** | Eyes focused, slight lean forward, sleeves rolled up |
| **Shocked** | Eyes wide, mouth slightly open, controlled surprise (NOT cartoonish) |
| **Satisfied** | Eyes half-closed, small smirk, beer in hand, lean back |
| **Exhausted** | Slouch, hand rubbing neck, dark circles hint |

### Outfit Overlay (thêm so với reference image)

| Chủ đề | Thêm |
|---|---|
| Bartending | Black vest, bar apron, cocktail shaker in hand |
| Food delivery | Delivery cap, bag strap over shoulder |
| DJ set | Headphones fully on, behind turntables |
| Office temp | Ill-fitting button-up, lanyard |
| Moving gig | T-shirt, work gloves, sweat |
| Pet-sitting | Leash in hand, overwhelmed look |

### Style Specs

```
- Sprunki character art style (2D stylized, vibrant colors, clean outlines)
- Bright, saturated, POPS on YouTube feed
- Simple background (gradient or single-color with contextual elements)
- 16:9 aspect ratio
- NO text in AI-generated image (text overlay thêm riêng)
- ADULT proportions and environment cues
```

### Text Overlay (thêm riêng)

```
- Font: Bold, modern sans-serif (not childish)
- Color: White + heavy black outline OR yellow + red outline
- 1 short punchy line: "$3.47 😭", "RENT'S DUE!", "HIRED! 🍸"
```

### Ví dụ Prompt

```
Prompt: Sprunki character art style, 16:9. Oren (orange Sprunki character with blue "O" jacket and headphones) stands on the left side of frame, looking at his phone screen with eyes wide and jaw clenched. The phone screen shows a banking app with "$3.47" balance. His other hand holds a crumpled rent notice. Simple apartment interior background — couch, records on wall, coffee table with takeout containers. Adult proportions, adult environment. Bright, colorful. No text.

Text overlay: "$3.47... RENT'S DUE?! 😱🏠"
```

## QUY TRÌNH TẠO BỘ 3

### Input cần
1. **WANT**: Oren muốn/cần gì? (rent, date night, new gear)
2. **HUSTLE**: Làm gì kiếm tiền?
3. **CLIMAX**: Kết quả bất ngờ
4. **KEY VISUAL**: Moment nào làm thumbnail?

### Output Format

```
## Title
😱 OMG — Oren Had $3.47 But RENT Was Due... His HUSTLE Was INSANE! 🍸💰🏠

## Thumbnail Prompt
[full prompt — no text, adult proportions]

## Thumbnail Text Overlay
"$3.47 → $$$?! 🤯"

## Description
[full 5-part English description]
```
